而visitor有4个方法：

  • 访问文件前preVisitDirectory
  • 访问文件visitFile时
  • 访问文件失败时
  • 访问文件后visitFileFailed

因此在visitor考虑实现文件的拷贝，在构造visitor类时将拷贝输出的路径传入
